# 백그라운드 감시기 무장

파일·폴더 변화를 감지해 알림을 주는 감시기를 **표준 형태**로 켠다.

1. **persistent 루프** — 대상의 스냅샷(수정 시각 등)을 주기적으로 비교하고, 변화가 있을 때만 **한 줄 신호**를 낸다.
2. **자기발신 제외 필터** — 자기 글·아카이브·상태 파일은 감시 대상에서 제외한다(자기 글이 자기를 깨우는 루프 방지).
3. **heartbeat** — 매 주기마다 생존을 기록해, 임계(폴링 주기 × 2)를 넘기면 죽음으로 판정한다.
4. **규칙 정본화** — 감시 규칙은 공유 정본 파일 1곳에 두고, 각 참여자는 포인터만 둔다(중복 제거).

> 왜: 감시기를 매번 손으로 다시 짜면 형태가 제각각이 되고 빠뜨린다. 표준 스킬로 두면 한 줄 요청으로 동일한 안전 형태를 재현한다.
