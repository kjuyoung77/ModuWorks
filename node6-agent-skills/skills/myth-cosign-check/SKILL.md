# 신화 콘텐츠 사실검증 (co-sign)

집필본의 사실 정합을 검증하고 **PASS / 교정**을 판정한다.

1. **귀속 대조** — 작품 크레딧(작가 · 연도 · 소장처)이 정확한지 대조한다.
2. **화가 환각 차단** — 존재하지 않는 작품을 실존처럼 쓰지 않았는지 확인한다(그 화가의 그 작품이 실제로 있는가).
3. **전거 층위 구분** — 원전(1차 사료)과 후대 판본을 구분하고, 특정 서술이 정전인지 후대 전승인지 명시한다.
4. **불확실 헷지** — 귀속이 확실치 않으면 「전(傳) / attributed」로 표기했는지 점검한다.
5. **판정** — PASS 또는 교정 목록(위치 + 근거)을 낸다.

> 왜: 신화·미술 콘텐츠는 매력적인 오류(가짜 귀속·후대 전승을 정전처럼)에 빠지기 쉽다. 이 체크리스트를 스킬로 고정하면 편마다 같은 기준으로 사실을 지킨다.
