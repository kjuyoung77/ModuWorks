# 도록 서사체 톤 변환

검증된 사실을 미술관 도록/도슨트의 **서사체**로 옮긴다.

1. **톤** — 차분한 학술·도록체. 클릭베이트·과장 금지.
2. **헷지 보존** — 귀속 불확실(전/attributed) 표기는 변환 후에도 유지한다.
3. **마크업 절제** — 강조 기호 남용 금지, 서사로 풀어 쓴다.
4. **표기 규칙** — 언어별 인용부호(「」 / « »), 연도 표기 규칙을 준수한다.
5. **구조** — 결론 단정이 아니라 서사(맥락 → 작품 → 여운)로 전개한다.

> 왜: 같은 사실도 톤이 흔들리면 브랜드가 깨진다. 톤 규칙을 스킬로 두면 누가 써도 도록의 목소리로 통일된다.
