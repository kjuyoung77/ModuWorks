# 세션 부트스트랩 (연속성·핸드오프)

새 세션/재개/재부팅 후 **첫 작업**으로 실행해, 이전 맥락을 안전하게 이어받는다.

1. **감시기 재무장** — 백그라운드 감시(메시지함·디렉터리 변화)를 다시 켠다.
2. **in-flight 점검** — 미처리 스레드를 색출한다. "회신요망=예" 표시를 검색하고, 발신 기록과 대조해 **아직 답이 안 온 스레드**를 목록화한다.
3. **맥락 복원** — 상태판·핸드오프 문서·메모리를 정독해 현재 상태를 복원한다.
4. **최신 결정 확인** — 가장 최근 결정 블록을 확인해 진행 방향을 맞춘다.
5. **요약** — 복원 결과를 3~5줄로 요약하고 "지금 할 일"을 제시한다.

> 왜: 세션이 끊길 때마다 맥락·미처리 건이 유실되기 쉽다. 이 절차를 스킬로 고정하면 매번 같은 순서로 안전하게 이어받는다.
