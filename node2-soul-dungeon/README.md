바이브코딩으로 만든 **소울나이트식 탑다운 던전 크롤러**. 순수 Canvas(외부 의존성 0).

## 조작
- **이동** WASD / 방향키 · **조준·발사** 마우스 클릭(꾹 누르면 연사)
- **상호작용** SPACE(포탈·NPC·옷장) · **일시정지** ESC

## 특징
- 3×3로 연결된 하나의 던전 — 4방향 통로로 인접 방 이동
- 전투 방은 입장 시 문이 잠기고, 몬스터를 다 처치하면 열림
- 방 타입 랜덤(전투·소굴·포션·무기·골드), 몬스터 4종, 미니맵
- 홈: 캐릭터 커스터마이징(옷장=색·의류상인=모자), NPC·가구
