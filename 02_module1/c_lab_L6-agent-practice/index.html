AI가 아는 Todo 앱

노드 6 실습 「Todo 앱을 'AI가 아는' 앱으로」 · 김주영

노드 5의 로컬 Todo 앱(SQLite)을 재료로, 범용 AI를 '내 앱을 아는 전용 조수'로 바꾼 실습. 방향 = 무거운 MCP가 아니라 앱에 CLI 달기 → 규칙 파일(CLAUDE.md)에 사용법 → 스킬로 반복 자동화.

산출물

파일역할
cli.jstodo.db·로직 재사용한 얇은 진입점. list·add·done·summary + 완료시각 컬럼 자동 보강
CLAUDE.md명령 사용법 + "todo.db 삭제 금지" 규칙. Claude Code가 자동 로드
todo-summary 스킬직접 데이터를 뒤지지 않고 cli.js summary를 대신 호출

실제 실행 (검증)

$ node cli.js list
[ ] 1. 노드5 실습 제출 준비하기  (~2026-07-25)  #공부 #급함
[x] 2. 우유 사기  #장보기

$ node cli.js add "노드6 스킬 과제 제출하기" --due 2026-07-22 --tag KDT
추가됨: 3. 노드6 스킬 과제 제출하기  (~2026-07-22)  #KDT

$ node cli.js done 3
완료 처리: 3번

$ node cli.js summary        # ← todo-summary 스킬이 대신 호출
# 오늘 완료 (2026-07-21) — 1건
- [완료] 노드6 스킬 과제 제출하기 (#3)

회고

범용 AI가 '내 앱을 아는 조수'로 바뀐 지점 = 규칙 파일이 CLI 사용법을 상시 알려주니 AI가 소스가 아니라 실제 데이터를 직접 읽고, 스킬이 그 CLI를 대신 불러 반복을 한마디로 압축한 것. 작은 개인 앱은 무거운 MCP 없이 'CLI + 짧은 스킬'로 충분히 조수화된다.
