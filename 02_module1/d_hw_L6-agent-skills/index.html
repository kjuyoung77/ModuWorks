AI 에이전트 스킬집 (5종)

노드 6 과제 「AI 에이전트 도구」 · 김주영

과제 요건은 "매번 반복하던 일을 스킬 하나로"지만, 반복 작업이 여러 개라 스킬집(5종)으로 만들었습니다. 각 스킬은 SKILL.md(머리말 name·description + 본문 절차) 하나로, AI가 관련 작업을 만나면 자동으로 꺼내 씁니다.

5종 — 무슨 불편을, 어떻게 없앴나

스킬계열불편 → 해결
session-bootstrap인프라세션 끊길 때마다 맥락·미처리 건 되짚음 → 재무장·in-flight 점검·상태 복원을 한 순서로 고정
irreversible-gate인프라push·배포·삭제를 무심코 실행 → 실행 직전 가역/비가역 판별 + 승인 게이트
watcher-arm인프라감시기를 매번 다르게 짜다 빠뜨림 → 표준 형태(자기발신 제외·heartbeat)로 재현
myth-cosign-check콘텐츠신화·미술 사실검증을 즉흥으로 → 귀속·환각·전거 층위·헷지 체크리스트로 고정
docent-tone콘텐츠도록 톤이 사람마다 흔들림 → 서사체 톤 규칙으로 통일

쓰는 법 (Claude Code)

skills/<이름>/ 폴더를 프로젝트의 .claude/skills/에 두면 자동 인식. 다른 대화에서 슬래시 명령으로 발동 — /session-bootstrap, /myth-cosign-check 등. (최상위 .claude/skills/ 최초 생성 직후엔 도구 재시작 필요.)

강건성 (코덱스 감사 반영 · 2026-07-22)

외부 감사관 검토로 실운영 수준 보강 — 오케스트레이터화(무조건 재무장 금지) · watcher 단일 인스턴스 lease 계약(PID·mtime 생존판정) · 위험등급 + 승인 SHA 결속 · 사실검증 재현 증거 + 사실/권리 분리. 검증 결과 → TESTS.md. ※ 스킬은 조언 계층이며 실제 차단은 permission/hook/branch protection 병행.

공개 저장소라 스킬은 재사용 가능한 일반 절차로만 담았습니다(키·비공개 URL·내부 세부 제외). 전체 내용은 GitHub 폴더 참고.
