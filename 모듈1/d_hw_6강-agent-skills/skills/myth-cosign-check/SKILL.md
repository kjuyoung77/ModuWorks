# 신화 콘텐츠 사실검증 (co-sign)

집필본의 사실 정합을 검증하고 **재현 가능한 증거**와 함께 판정한다.

## 점검 항목
1. **귀속 대조** — 작품 크레딧(작가 · 연도 · 소장처)이 정확한가.
2. **화가 환각 차단** — 존재하지 않는 작품을 실존처럼 쓰지 않았는가(그 화가의 그 작품이 실제로 있는가).
3. **전거 층위 구분** — 원전(1차 사료)과 후대 판본을 구분, 정전/후대 전승 명시.
4. **불확실 헷지** — 귀속이 확실치 않으면 「전(傳) / attributed」로 표기.

## 판정 형식 (재현 가능 증거)
각 판정에 다음을 남긴다 — **검증 대상 위치 + 근거(출처 URL / 문헌 위치) + 접근일 + 근거 요약 + 확신도.**
(예: `나르키소스 귀속=카라바조(전) | 근거: [소장처 카탈로그 URL] | 접근 2026-07-22 | "attributed to Caravaggio" 표기 확인 | 확신도 중`)

## ★사실과 권리 분리
판정을 둘로 나눈다:
- **FACT PASS / FAIL** — 작가·작품·전거의 사실 정합.
- **RIGHTS NOT ASSESSED** — 이미지·상업 이용 권리는 **별도**. 작품명·작가·소장처가 맞아도 **상업 이용 권리가 자동 통과되지 않는다.**

> 왜: "맞다"는 판정이 근거 없이 떠다니면 재검증이 안 되고, 사실 정합을 권리 통과로 오해하기 쉽다. 증거 형식과 사실/권리 분리를 스킬로 고정한다.
