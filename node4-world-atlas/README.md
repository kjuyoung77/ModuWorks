---

## 백엔드 (Supabase · RLS 보안)
- **저장·집계 기능** — 방문자 수 · 나라별 조회수(인기순위) · 댓글 · 가봤어요/가고싶어요 · 즐겨찾기
- **인가** — 익명 로그인 → `auth.uid()` 로 본인 것만 수정
- **프라이버시** — 방문 로그 원문은 SELECT 차단, 집계는 `SECURITY DEFINER` 함수로만 노출
- **비밀값** — 공개(publishable) 키만 프론트에, service_role 비밀키 미사용 · 데이터는 RLS가 보호

## 화면
- **한/영 토글(i18n)** — UI·26개국 데이터·명소 해설(ko/en 위키백과 자동 전환)
- 위키백과 랜드마크 히어로 연동, 목록 검색·지역 필터·즐겨찾기·인기순 정렬

## 파일 안내
| 파일 | 내용 |
|---|---|
| `index.html` | 목록(검색·지역·즐겨찾기·인기순·방문자 수) |
| `country.html` | 나라별 상세(위키 히어로·개요·나의 기록·댓글·조회수) |
| `data.js` | 26개국 데이터 + i18n 사전 + Supabase 설정·헬퍼 |
| `SUPABASE_설정.sql` | 테이블 + RLS 정책 + 집계 함수(RPC) |
| `제출_런북.md` | 설계 한 장 + 셋업 절차 + 보안 체크리스트 |
| `flags/` | 국기 SVG 26개 |
