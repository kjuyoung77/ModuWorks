## 게임 소개
Canvas로 좌·우 두 장면을 직접 그려 **다른 곳 5개**를 찾는 게임.

- 5스테이지: 밤 캠핑장 → 낮 호수 → 협곡 노을 → 은하수 밤 → 새벽 안개
- 제한시간(숫자+바), 클리어 시 +8초 보너스, 오답 −2초
- 스테이지가 오를수록 클릭 판정 축소·배경 복잡도 증가
- **ESC 일시정지**(정지 중 그림은 가려짐 — 엿보기 방지), 클리어 시 홈 복귀
